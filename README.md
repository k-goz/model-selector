## 功能

- 🔍 **实时价格对比** — $/1M tokens / ¥元/1M tokens
- 🏷️ **平台筛选** — 7 大平台一键切换
- 🏷️ **场景筛选** — 日常对话 / 深度推理 / 视觉图片 / 图片生成 / 视频生成
- 📋 **一键复制** — 点击卡片复制 API 接入命令
- ⌨️ **键盘快捷键** — `/` 聚焦搜索，`Esc` 退出
- 🎨 **价格颜色分级** — 免费(绿) / 极便宜(蓝) / 中等(黄) / 贵(红) / 极贵(紫)
