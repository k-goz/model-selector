## 功能

### 核心功能
- **多平台聚合** — 13 家平台 1101 个模型统一呈现
- **实时价格对比** — 支持 CNY / USD 双货币切换
- **一键复制** — 点击卡片复制 API Base URL，带 Toast 成功提示
- **Base URL 显示** — 每个卡片直接展示 API 接入地址

### 筛选与排序
- **平台筛选** — 13 家平台一键切换，显示各平台模型数
- **价格分级筛选** — 免费 / 便宜 / 中等 / 贵 / 极贵
- **场景筛选** — 日常对话 / 深度推理 / 视觉图片 / 图片生成 / 视频生成 / 编程代码
- **标签筛选** — 免费 / 旗舰 / 视觉 / 推理 / 长上下文 / 开源等
- **上下文长度筛选** — 按上下文窗口大小过滤
- **价格区间筛选** — 自定义输入/输出价格范围
- **多维排序** — 默认 / 输入价升序 / 输入价降序 / 输出价升序 / 输出价降序 / 名称
- **模糊搜索** — 200ms 防抖，按 / 快速聚焦

### 智能功能
- **智能推荐** — 日常对话 / 编程 / 推理 / 视觉场景一键推荐最优模型
- **跨平台比价** — 精确模型名匹配，同一模型在不同平台的价格对比，显示最低价和溢价百分比
- **月费计算器** — 输入对话次数 / Token 数 / 输出输入比，计算各模型月费排名
- **预算反推** — 输入月预算，反推每个模型可用对话次数
- **模型对比** — 最多勾选 3 个模型并排对比

### 体验
- **暗色模式** — 默认 Linear Aesthetic 暗色精度美学，可切换亮色
- **收藏功能** — 收藏常用模型，本地持久化
- **列表/网格视图** — 两种浏览模式切换
- **筛选持久化** — 筛选条件通过 URL hash 持久化，可分享
- **移动端适配** — 768px / 400px 两级响应式，筛选栏横向滚动
- **键盘快捷键** — `/` 搜索 / `Esc` 清空 / `D` 暗色 / `V` 视图 / `1-9` 切换平台

## 价格说明

- **OpenRouter**：显示原始美元价格（$/1M tokens）
- **国内平台**：显示人民币价格（元/1M tokens）
- 阿里百炼价格从 API 实时获取，其余平台使用价格对照表（需定期更新）
- 标注「价格待确认」的模型请至平台控制台核实

## 技术架构

- 纯 HTML + CSS + JS，零依赖，单文件可离线使用
- Python 数据抓取脚本（`generate.py`），从各平台 API 拉取模型数据生成静态 HTML
- CSS 变量体系 + Linear Aesthetic 设计风格（shimmer / glow / micro-border / glassmorphism）
- API Key 通过环境变量注入，支持本地缓存和重试
- 部署于 Vercel（静态站点）
