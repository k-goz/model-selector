隐私政策

最后更新：2026年5月8日

1. 概述

AI 模型选择器（"我们"）尊重您的隐私。本政策说明我们收集什么数据、如何使用、以及您的权利。本站托管于 Vercel，开源代码在 GitHub。

2. 我们收集的数据

  • 本地存储：我们将您的偏好（暗色模式、收藏、列表视图）存储在浏览器 localStorage 中，数据不会离开您的浏览器。
  • 无用户账号：我们不需要注册，不收集个人信息。
  • 无追踪：我们不使用分析 Cookie 或追踪个人用户。

3. 第三方服务

  • Google AdSense：可能投放广告并使用 Cookie 进行广告个性化。详见 Google 隐私政策。
  • Vercel 托管：托管方可能收集服务器日志（IP、浏览器、URL）。详见 Vercel 隐私政策。

4. API 数据

我们从公共 API 和官方定价页获取 AI 模型定价数据。我们不会将您的数据发送到任何 AI 平台。所有 API 调用均在服务端站点生成时完成，而非从您的浏览器发出。

5. Cookie

我们不设置第一方 Cookie。Google AdSense 可能设置第三方 Cookie 用于广告投放。您可以通过 Google 退出工具选择退出。

6. 数据保留

我们保留定价数据用于交叉验证和趋势检测。不保留任何个人数据，因为我们不收集任何个人数据。

7. 您的权利

由于我们不收集个人数据，因此没有数据需要访问、删除或转移。您可以随时通过浏览器设置清除 localStorage。

8. 变更

我们可能更新本政策。变更将在此处发布并更新日期。继续使用即视为接受。

9. 联系方式

如有问题，请在 GitHub 提交 Issue。
