Data Sources

Pricing data from multiple sources, prioritized for accuracy:

  • A — API Real-time: Directly from platform APIs, always current
  • S — Official Scraping: Parsed from official pricing pages
  • DB — Price Database: Curated local database with prefix matching
  • L — LiteLLM Community: Open-source pricing data as fallback
  • P — Proxy Pricing: Self-hosted proxy platform pricing
