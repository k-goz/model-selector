关于 AI 模型选择器

全网最全面的实时 AI 模型定价对比工具，帮助开发者找到最适合需求和预算的 AI API。

我们做什么

AI 模型选择器从 25+ AI 平台实时聚合定价和能力数据，让你随时掌握 AI 市场全景，不用再逐个翻看几十个定价页。

核心功能

💰 价格对比

所有平台并排定价，支持人民币和美元

🔍 跨平台比价

同一模型在不同平台的不同价格一目了然

⚡ 接口测速

实时测量任意模型在各平台的 TTFB 延迟

💻 一键接入

复制 Python、cURL 等即用 API 命令

🔎 Token 计价器

粘贴真实文本，自动计算各平台花费

🎯 智能筛选

按价格、上下文、标签、模型家族筛选

数据来源

定价数据来自多个来源，按优先级排序：

  • A — API 实时：直接从平台 API 获取，始终最新
  • S — 官方爬取：解析官方定价页面
  • DB — 价格数据库：本地策划数据库，前缀匹配
  • L — LiteLLM 社区：开源定价数据兜底
  • P — 代理定价：自营代理平台定价
